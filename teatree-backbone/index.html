Welcome to Backbone Tea Tree

This is one client-side MVC & single-page applications (SPAs) based on Backbone.js with following features:

  1. one MVC implementation on Backbone framework
  2. when one Model's data change, all the Views bound to that Model automatically re-render
  3. integrate with RESTful services based on Slim framework (PHP version)
  4. one Responsive Web Design (RWD) application by CSS3 Media Queries
